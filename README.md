# Natural Language Scripts

My natural language scripts which are for LLM agents

Just found that if you rich enough (or have free APIs), you don't need to maintain complex setup scripts for all different OS, machine, devices etc.
Just let LLM do it for you. And save your energy to where worth to go to.
